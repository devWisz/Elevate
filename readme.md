# Elevate

A high-performance concurrent website status checker built with Go.

Release link : (You can download the .exe file and run on your local computer) : 


## Description
Elevate is a lightweight CLI utility designed to demonstrate the power of Go's concurrency model. It uses **Goroutines** to check multiple URLs at the same time and **Channels** to aggregate the results, making it significantly faster than traditional sequential scripts.

## Details
- **Language:** Go (Golang)
- **Concurrency:** Goroutines & Channels
- **Dependencies:** None (Standard Library only)
- **Timeout:** 5-second request limit per site

## Setup & How to Run
1. **Install Go:** Make sure you have Go installed on your system.
2. **Download the code:** Save the script as `main.go`.
